# Анализ и исправление структуры отображения таблицы

Необходимо проанализировать загрузку таблицы и правильную расстановку ячеек на основе данных из `D:\1C\BASE\src\cf\Documents\Сборка\Templates\ПечатьСборки\Ext`. Выровнять структуру отображения именованных областей, нумерации строк, колонок, ячеек.

## Анализ проблем

### 1. Анализ структуры данных из XML файла

- **Действие:** Изучить XML файл из указанной папки
- **Цель:** Понять структуру именованных областей (namedItem), строк (rowsItem), колонок (columns)
- **Что проверить:**
- Количество строк и их индексы
- Количество колонок и их индексы
- Именованные области и их координаты (beginRow, endRow, beginColumn, endColumn)
- Типы именованных областей (Rows, Columns, Rectangle)

### 2. Анализ текущей логики рендеринга

- **Файл:** `src/webview/components/TemplateEditor/TemplateTable.tsx`
- **Что проверить:**
- Как вычисляется `maxColumns` - используется ли правильный способ подсчета максимального количества колонок
- Как отображаются строки - правильно ли используется индекс из `rowsItem`
- Как отображаются колонки - правильно ли используются индексы колонок
- Соответствие между индексами в данных (rowsItem[].row.c[].i) и отображаемыми колонками (Array.from({ length: maxColumns }))

### 3. Проверка функций работы с именованными областями

- **Файл:** `src/utils/templateUtils.ts`
- **Что проверить:**
- Правильность работы `getNamedAreaForRow` - корректно ли определяются именованные области для строк
- Правильность работы `getNamedAreaForColumn` - корректно ли определяются именованные области для колонок
- Правильность работы `isCellOnNamedAreaBoundary` - корректно ли определяются границы
- Соответствие индексов в именованных областях (beginRow, endRow, beginColumn, endColumn) и реальных индексов строк/колонок

### 4. Анализ структуры HTML таблицы

- **Файл:** `src/webview/components/TemplateEditor/TemplateTable.tsx`
- **Что проверить:**
- Правильность структуры thead (количество строк, порядок колонок)
- Правильность структуры tbody (порядок колонок в каждой строке)
- Соответствие количества колонок между thead и tbody

### 5. Проверка CSS позиционирования

- **Файл:** `src/webview/components/TemplateEditor/template-editor.css`
- **Что проверить:**
- Правильность позиционирования sticky элементов
- Правильность ширины колонок (named-area-header: 80px, row-header: 40px, column-header: 100px)
- Правильность высоты строк заголовков

## План исправлений

После анализа данных определить конкретные проблемы и исправить:

1. Выравнивание колонок (если есть смещение между thead и tbody)
2. Правильное отображение именованных областей (если они не соответствуют данным)
