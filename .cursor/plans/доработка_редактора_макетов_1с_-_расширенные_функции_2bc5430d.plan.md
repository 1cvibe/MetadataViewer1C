#Доработка редактора макетов 1С - расширенные функции

## Выявленные недостающие элементы структуры

На основе изучения макета `ПФ_MXL_СправкаПоОтпускамСотрудника` выявлены следующие недостающие элементы:

### 1. Расширение интерфейсов для форматов

**Файл**: [src/templatInterfaces.ts](src/templatInterfaces.ts)**Недостающие поля в `TemplateFormat`:**

- `height` - высота строки
- `verticalAlignment` - вертикальное выравнивание ("Top", "Bottom", "Center")
- `fillType` - тип заполнения ("Parameter", "Text")
- `indent` - отступ
- `autoIndent` - автоматический отступ
- `textColor` - цвет текста (может быть стиль: "style:NegativeTextColor")
- `backColor` - цвет фона (может быть стиль: "style:ToolTipBackColor")
- `format` - вложенный формат для чисел/дат (структура с v8:item/v8:content)
- `markNegatives` - отметка отрицательных чисел (true/false)

**Недостающие поля в `TemplateCellData`:**

- `detailParameter` - параметр расшифровки (элемент `<detailParameter>`)
- `note` - примечание к ячейке (сложная структура с координатами, текстом, форматом)

**Недостающие поля в `TemplateRow.row`:**

- `empty` - признак пустой строки (элемент `<empty>` со значением true)

**Обновление `TemplateCell`:**

- `i` должен быть опциональным (индекс определяется порядком, если отсутствует)

**Обновление `TemplateRow.row`:**

- `columnsID` должен быть опциональным (может отсутствовать)
- `formatIndex` должен быть опциональным

### 2. Обновление парсинга в metadataView.ts

**Файл**: [src/metadataView.ts](src/metadataView.ts)**Необходимо добавить в `arrayPaths`:**

- `'document.format'` - если format может быть массивом (уже есть, но проверить)
- `'document.font'` - если font может быть массивом (уже есть, но проверить)
- `'document.merge'` - если merge может быть массивом (уже есть, но проверить)

**Обновление парсинга для:**

- Пустых строк (`<empty>true</empty>`)
- Ячеек без атрибута `i` (индекс определяется порядком)
- Примечаний к ячейкам (`<note>`)
- Детальных параметров (`<detailParameter>`)
- Вложенных форматов для чисел/дат (`<format><v8:item>...</v8:item></format>`)
- Цветов, которые могут ссылаться на стили (`style:NegativeTextColor`)

### 3. Реализация полного сохранения в templateXmlSaver.ts

**Файл**: [src/utils/templateXmlSaver.ts](src/utils/templateXmlSaver.ts)**Реализовать функции:**

- `updateFormatsInDom` - обновление форматов (format)
- `updateFontsInDom` - обновление шрифтов (font)
- `updateColumnsInDom` - обновление колонок (columns)
- `updateNotesInDom` - обновление примечаний к ячейкам (note)
- `updateDetailParametersInDom` - обновление детальных параметров (detailParameter)
- `updateCellFormatInDom` - обновление формата ячейки с учетом всех полей

**Особенности сохранения:**

- Сохранение вложенных форматов для чисел/дат
- Сохранение цветов (включая ссылки на стили)
- Сохранение пустых строк (`<empty>true</empty>`)
- Сохранение ячеек без атрибута `i` (если индекс определяется порядком)
- Сохранение объединенных ячеек без `h` (если высота не задана)
