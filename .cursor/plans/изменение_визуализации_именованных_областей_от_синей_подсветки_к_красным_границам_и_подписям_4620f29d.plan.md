# Изменение визуализации именованных областей: от синей подсветки к красным границам и подписям

Изменяем визуализацию именованных областей в редакторе макетов: убираем синюю подсветку ячеек, добавляем подписи именованных строк и колонок рядом с номерами, добавляем красные границы только на граничных ячейках именованных областей (как в EDT).

## Изменения

### 1. Создание функций для работы с именованными областями строк/колонок

- **Файл:** [`src/utils/templateUtils.ts`](src/utils/templateUtils.ts)
- Добавить функцию `getNamedAreaForRow(template: TemplateDocument, rowIndex: number): NamedArea | null` — возвращает именованную область типа "Rows" для указанной строки (если строка входит в именованную область)
- Добавить функцию `getNamedAreaForColumn(template: TemplateDocument, colIndex: number): NamedArea | null` — возвращает именованную область типа "Columns" для указанной колонки
- Добавить функцию `isCellOnNamedAreaBoundary(template: TemplateDocument, rowIndex: number, colIndex: number, area: NamedArea): { top: boolean, bottom: boolean, left: boolean, right: boolean }` — определяет, находится ли ячейка на границе именованной области

### 2. Обновление отображения заголовков строк

- **Файл:** [`src/webview/components/TemplateEditor/TemplateTable.tsx`](src/webview/components/TemplateEditor/TemplateTable.tsx)
- В рендеринге заголовков строк (около строки 384-390) получить именованную область для текущей строки через `getNamedAreaForRow`
- Отображать имя области слева от номера строки (например, "Заголовок | 1" или "Заголовок" в отдельной части заголовка)
- Использовать структуру: `<td className="template-table-row-header"><span className="named-area-label">...</span><span className="row-number">...</span></td>`

### 3. Обновление отображения заголовков колонок

## Детали реализации

### Функция `getNamedAreaForRow`

- Перебирает все именованные области типа "Rows"
- Проверяет, входит ли `rowIndex` в диапазон `[area.startRow, area.endRow]`
- Возвращает первую найденную область или null

### Функция `getNamedAreaForColumn`

- Перебирает все именованные области типа "Columns"
- Проверяет, входит ли `colIndex` в диапазон `[area.startCol, area.endCol]`
- Возвращает первую найденную область или null

### Функция `isCellOnNamedAreaBoundary`

- Определяет, является ли ячейка граничной для именованной области
- Для типа "Rows": верхняя граница — `rowIndex === area.startRow`, нижняя — `rowIndex === area.endRow`, левая/правая — первая/последняя колонка
- Для типа "Columns": левая граница — `colIndex === area.startCol`, правая — `colIndex === area.endCol`, верхняя/нижняя — первая/последняя строка
- Возвращает объект с флагами `{ top, bottom, left, right }`

### Отображение подписей

- Для строк: подпись слева от номера строки, выравнивание по левому краю
- Для колонок: подпись под/над буквой колонки, выравнивание по центру
- Использовать flexbox для расположения элементов

### Красные границы

- Применять только к граничным ячейкам именованных областей
- Использовать `border-top`, `border-bottom`, `border-left`, `border-right` с красным цветом
