# Реализация именованных строк и колонок

## Обзор

Добавление функциональности для создания именованных строк и колонок:

- Именованные строки: `area.type = "Rows"`, `beginRow/endRow` - конкретные номера, `beginColumn/endColumn = -1`
- Именованные колонки: `area.type = "Columns"`, `beginColumn/endColumn` - конкретные номера, `beginRow/endRow = -1`
- Элемент `namedItem` всегда имеет атрибут `xsi:type="NamedItemCells"`

## Изменения в интерфейсах

### [src/templatInterfaces.ts](src/templatInterfaces.ts)

- Обновить `NamedItemArea.type` для поддержки типов `"Rows"` и `"Columns"` (уже есть в коде)
- Убедиться, что комментарии корректны для -1 значений

## Изменения в утилитах

### [src/utils/templateUtils.ts](src/utils/templateUtils.ts)

1. **Обновить `createNamedArea`**:

- Добавить поддержку типов `"Rows"` и `"Columns"`
- Для типа `"Rows"`: автоматически устанавливать `beginColumn/endColumn = -1`
- Для типа `"Columns"`: автоматически устанавливать `beginRow/endRow = -1`

2. **Добавить функцию `createNamedRows`**:

- Принимает диапазон строк (startRow, endRow)
- Создает именованную область с типом `"Rows"`, координаты колонок = -1
- Автоматически генерирует имя или использует предложенное

3. **Добавить функцию `createNamedColumns`**:

- Принимает диапазон колонок (startCol, endCol)
- Создает именованную область с типом `"Columns"`, координаты строк = -1
- Автоматически генерирует имя или использует предложенное

4. **Обновить `findNamedAreaByPosition`**:

- Учитывать типы `"Rows"` (все колонки) и `"Columns"` (все строки)
- Для `"Rows"`: проверять только номера строк, игнорировать колонки
- Для `"Columns"`: проверять только номера колонок, игнорировать строки

5. **Обновить валидацию**:

- При типе `"Rows"`: проверять, что `beginColumn/endColumn = -1`
- При типе `"Columns"`: проверять, что `beginRow/endRow = -1`
