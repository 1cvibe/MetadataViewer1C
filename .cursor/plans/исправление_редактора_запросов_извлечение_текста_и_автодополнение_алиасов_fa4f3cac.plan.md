# Исправление ре

дактора запросов: извлечение текста и автодополнение алиасов

## Проблема 1: Неполное отображение текста запроса

**Причина**: В BSL многострочные строки используют двойные кавычки `""` для экранирования кавычек внутри строки, но текущий код ищет только `\"`. При поиске закрывающей кавычки алгоритм находит первую `"`, которая может быть частью экранированной последовательности `""`.**Решение**: Исправить логику поиска кавычек в `src/autogen-bsl/init.ts`:

- При поиске закрывающей кавычки проверять, не является ли она частью последовательности `""` (экранированная кавычка)
- Если найдена `""`, пропускать её и продолжать поиск
- Учесть, что в BSL `""` внутри строки означает одну кавычку, а не конец строки

## Проблема 2: Автодополнение не работает для алиасов справочников и документов

**Причина**:

- Функция `resolveAliasTableKey` в `src/webview/components/DcsEditor/QueryEditorEnhanced.tsx` ищет только регистры (РегистрНакопления, РегистрСведений, РегистрБухгалтерии, РегистрРасчета)
- В `buildQueryContext` в `src/webview/utils/monacoQueryLanguage.ts` для разрешения алиасов используется только `aliasMap`, который строится на основе токенов, но не учитывает метаданные для справочников и документов

**Решение**:

1. Расширить `resolveAliasTableKey` для поддержки всех типов метаданных:

- Справочник
- Документ
- Перечисление
- ПланВидовХарактеристик
- ПланВидовРасчета
- ПланСчетов
- БизнесПроцесс
- Задача
- ПланОбмена
- Константа
- ЖурналДокументов

2. Использовать `resolveAliasTableKey` в `buildQueryContext` для разрешения алиасов, когда `aliasMap` не содержит нужного алиаса
3. Обновить регулярное выражение в `resolveAliasTableKey` для поиска всех типов метаданных

## Файлы для изменения

1. **src/autogen-bsl/init.ts** (строки 152-174):

- Исправить логику поиска закрывающей кавычки с учетом экранирования `""`

2. **src/webview/components/DcsEditor/QueryEditorEnhanced.tsx** (строки 24-47):

## Детали реализации

### Исправление извлечения текста запроса

В BSL экранирование кавычек работает так:

- `"` - начало/конец строки
- `""` - одна кавычка внутри строки
- `\"` - экранированная кавычка (если используется обратный слэш)

Нужно при поиске закрывающей кавычки проверять:

```typescript
if (fullText[i] === '"') {
  // Проверяем, не является ли это экранированной кавычкой ""
  if (i + 1 < fullText.length && fullText[i + 1] === '"') {
    // Это экранированная кавычка "", пропускаем оба символа
    i += 2;
    continue;
  }
  // Это настоящая закрывающая кавычка
  closeQuoteOffset = i;
  break;
}
```



### Расширение автодополнения для алиасов

В `resolveAliasTableKey` нужно добавить все типы метаданных:
