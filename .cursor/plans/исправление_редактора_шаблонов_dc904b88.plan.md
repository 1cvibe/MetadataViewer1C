# Исправ

ление редактора шаблонов

## Проблемы и решения

### 1. Кнопка "Удалить" для именованных областей

**Проблема:** После удаления именованной области не обновляется отображение таблицы (заголовки и красная сетка).**Решение:**

- Обработчик `handleDeleteNamedArea` в `TemplateEditorApp.tsx` уже вызывает `setTemplateDocument`, что должно обновить таблицу
- Проверить, что `deleteNamedArea` корректно удаляет область из `namedItem`
- Убедиться, что после удаления происходит перерисовка таблицы через React state

**Файлы:**

- `src/webview/components/TemplateEditor/TemplateEditorApp.tsx` - обработчик уже есть, проверить работу
- `src/utils/templateUtils.ts` - функция `deleteNamedArea` уже реализована

### 2. Вложенные именованные области не выводятся

**Проблема:** Вложенные области (например, "ДанныеЦены" внутри "Данные") не отображаются в подписях.**Решение:**

- Функции `getNamedAreasForRow` и `getNamedAreasForColumn` уже созданы и исключают Rectangle
- Проверить, что `findNamedAreaByPosition` также исключает Rectangle при определении границ
- Убедиться, что в `TemplateTable.tsx` используется правильная логика для отображения всех областей

**Файлы:**

- `src/utils/templateUtils.ts` - обновить `findNamedAreaByPosition` для исключения Rectangle при определении границ (но оставить для общего поиска)
- `src/webview/components/TemplateEditor/TemplateTable.tsx` - проверить использование `getNamedAreasForRow` и `getNamedAreasForColumn`

### 3. Красная сетка для колонок формируется неправильно

**Проблема:** Красные границы для именованных колонок отображаются не на первой и последней колонке области.**Решение:**

- Проверить логику в `isCellOnNamedAreaBoundary` для типа `Columns`
- Для типа `Columns`: границы должны быть только на первой (`startCol`) и последней (`endCol`) колонке области
- Верхняя и нижняя границы должны быть на всех строках, но только на первой и последней колонке

**Файлы:**

- `src/utils/templateUtils.ts` - исправить логику в `isCellOnNamedAreaBoundary` для типа `Columns`

### 4. Кнопки Объединить/Разъединить ячейки не работают

**Проблема:** Кнопки не активны или не выполняют действие.**Решение:**

- Проверить условия активации: `onMergeCells` требует `selectedRange`, `onUnmergeCells` требует `selectedCell`
- Убедиться, что обработчики правильно вызывают функции `mergeCells` и `unmergeCells`
- Проверить, что после объединения/разъединения обновляется состояние таблицы

**Файлы:**

- `src/webview/components/TemplateEditor/TemplateEditorApp.tsx` - проверить обработчики `handleMergeCells` и `handleUnmergeCells`
- `src/webview/components/TemplateEditor/TemplateToolbar.tsx` - проверить передачу обработчиков

### 5. Добавить кнопку "Показать/Скрыть красную сетку именованных областей"

**Проблема:** Отсутствует возможность скрыть красные границы именованных областей.**Решение:**

- Добавить состояние `showNamedAreaBorders` в `TemplateEditorApp.tsx`
- Добавить кнопку в `TemplateToolbar.tsx` для переключения
- Передать состояние в `TemplateTable.tsx` и условно применять классы границ

**Файлы:**

- `src/webview/components/TemplateEditor/TemplateEditorApp.tsx` - добавить состояние и обработчик
- `src/webview/components/TemplateEditor/TemplateToolbar.tsx` - добавить кнопку
- `src/webview/components/TemplateEditor/TemplateTable.tsx` - условно применять классы границ

### 6. Кнопки Жирный/Курсив/Подчеркнутый не работают

**Проблема:** Обработчики не передаются в `TemplateToolbar`.**Решение:**

- Использовать функцию `updateCellFont` для обновления шрифта
- Передать обработчики в `TemplateToolbar`

**Файлы:**
