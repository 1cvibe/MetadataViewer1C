# Обзор проекта MetaDataViewer_v3

## Назначение проекта
VS Code расширение для работы с метаданными конфигураций 1С:Предприятие 8. Позволяет просматривать, редактировать и навигироваться по структуре конфигураций 1С в форматах XML и EDT.

## Технологический стек
- **Язык**: TypeScript (ES2020)
- **Платформа**: VS Code Extension API (v1.56.0+)
- **Frontend**: React 18.2.0, React DOM 18.2.0
- **Редактор кода**: Monaco Editor 0.44.0
- **XML парсинг**: @xmldom/xmldom 0.8.11, fast-xml-parser 4.0.11
- **Сборка**: Webpack 5.88.2, TypeScript 4.8.4
- **Стилизация**: CSS, VS Code Theme API

## Основные возможности
- Дерево метаданных конфигурации 1С в панели VS Code
- Открытие модулей BSL через контекстное меню
- Редактирование свойств объектов метаданных
- Просмотр и редактирование форм
- Редактор СКД (Data Composition Schema)
- Генерация BSL кода из комментариев `// @gen`
- Закладки в BSL модулях (Alt+F2, F2)
- Управление отступами (Tab, Alt+Backspace)

## Структура проекта
- `src/` - исходный код TypeScript
  - `extension.ts` - точка входа расширения
  - `webview/` - React компоненты для webview панелей
  - `autogen-bsl/` - генерация BSL кода
  - `xmlParsers/` - парсеры XML конфигураций
  - `utils/` - утилиты (закладки, логирование, работа с XML)
  - `panels/` - панели редакторов (Metadata, Template)
- `out/` - скомпилированный JavaScript
- `media/` - собранные bundle файлы для webview
- `test-cases/` - тестовые файлы и скрипты

## Форматы конфигураций
- **XML формат**: ConfigDumpInfo.xml + Configuration.xml
- **EDT формат**: Configuration.mdo

## Язык интерфейса
Русский язык по умолчанию для всех команд и интерфейсных элементов.
