# Команда "Переиндексировать структуру"

## Назначение
Команда `metadataViewer.reindexStructure` выполняет полную пересборку дерева метаданных конфигураций 1С и очистку всего кэша.

## Реализация
Файл: `src/metadataView.ts`, метод `reindexStructure()` (строки 176-193)

## Что делает команда

### 1. Очистка всего кэша (L1, L2, L3)
- **L1**: in-memory кэш текущей сессии (Map)
- **L2**: workspaceState (указатели на L3 файлы)
- **L3**: JSON файлы на диске в `globalStorageUri`
- Вызывает `cache.invalidateAll()` из `MetadataCache`

### 2. Очистка дерева
- Удаляет все дочерние элементы корневого узла "Конфигурации"
- Обновляет провайдер дерева через `dataProvider.update()`

### 3. Пересканирование конфигураций
- Вызывает `LoadAndParseConfigurationXml()` для каждой папки workspace
- Ищет файлы конфигураций:
  - `**/ConfigDumpInfo.xml` и `**/Configuration.xml` (XML формат)
  - `**/Configuration/Configuration.mdo` (EDT формат)
- Глубина поиска: настройка `metadataViewer.searchDepth` (по умолчанию 3)
- Парсит найденные конфигурации и добавляет их в дерево заново

## Когда использовать
- После изменений в файловой структуре конфигураций
- Если дерево метаданных отображается некорректно
- После добавления/удаления конфигураций в workspace
- При проблемах с кэшем (устаревшие данные)

## Доступ к команде
- **Контекстное меню**: правый клик на корневом элементе "Конфигурации" в панели "1C Metadata Viewer"
- **Условие видимости**: `viewItem == main` (только для корневого элемента)
- **Группа меню**: `metadata`

## Производительность
- Может занять время для больших конфигураций
- Выполняется асинхронно, не блокирует UI
- Логирование в Output панели "1C Metadata Viewer"

## Связанные компоненты
- `MetadataCache.invalidateAll()` - очистка кэша
- `LoadAndParseConfigurationXml()` - загрузка и парсинг конфигураций
- `tree[0].children` - корневой элемент дерева метаданных
- `dataProvider` - провайдер данных для TreeView

## Важно
Команда полностью пересобирает дерево с нуля, что полезно при проблемах с кэшем или после структурных изменений в конфигурациях.
