# Редактор макетов табличных документов 1С

## Основные возможности редактора макетов (из документации 1С)

### 1. Редактирование текста
- Ввод текста в ячейки
- Редактирование существующего текста (Enter или F2)
- Многострочный текст (Shift+Enter)
- Отмена/возврат действий (Правка → Отмена/Вернуть)

### 2. Управление структурой
- Вставка строк выше/ниже текущей
- Удаление строк
- Вставка колонок слева/справа
- Удаление колонок
- Вставка/удаление отдельных ячеек (с раздвижением)
- Объединение/разъединение ячеек

### 3. Изменение размеров
- Изменение ширины колонок вручную (перетаскивание границы)
- Автоподбор ширины (двойной клик на границе)
- Изменение высоты строк вручную
- Изменение размеров с клавишей Ctrl

### 4. Форматирование ячеек
- Шрифт (имя, размер, жирный, курсив, подчеркивание, зачеркивание)
- Выравнивание (левое, по центру, правое, по ширине)
- Границы (все, левая, правая, верхняя, нижняя, без границ)
- Цвет текста и фона
- Размещение текста (обычное, перенос по словам)

### 5. Дополнительные функции
- Именованные области (присвоение имен диапазонам ячеек)
- Фиксация строк/колонок (заморозка при прокрутке)
- Режимы отображения:
  - Сетка
  - Заголовки строк/колонок
  - Примечания
  - Черно-белый просмотр
  - Режим просмотра страниц
  - Масштабирование (Ctrl + колесо мыши)
- Защита от редактирования

### 6. Параметры (заполнители)
- Вставка параметров в ячейки (например, `<Параметр>`)
- Трансляция параметров при построении отчета

### 7. Свойства табличного документа
- Макет (флаг, что это макет)
- Итоги по горизонтали/вертикали
- Картинка фона
- Фиксированный фон
- Переход после ввода (Enter)
- Область печати

## Структура макета (XML)

### Структура файла Template.xml
- `document` - корневой элемент макета
  - `columns` - массив групп колонок
    - `id` - идентификатор группы (опционально)
    - `size` - размер (количество колонок)
    - `columnsItem` - массив определений колонок
      - `index` - индекс колонки
      - `column.formatIndex` - ссылка на формат
  - `rowsItem` - массив строк
    - `index` - индекс строки (начинается с 1)
    - `row` - данные строки
      - `columnsID` - идентификатор группы колонок (опционально)
      - `formatIndex` - ссылка на формат строки
      - `c` - массив ячеек
        - `i` - индекс колонки ячейки
        - `c` - данные ячейки
          - `f` - ссылка на формат (индекс в массиве format)
          - `parameter` - имя параметра (заполнителя), если есть
          - `tl` - текстовое содержимое
            - `v8:item.v8:content` - текст ячейки
  - `format` - массив форматов ячеек
    - `width` - ширина колонки (для форматов колонок)
    - `horizontalAlignment` - выравнивание (Left, Center, Right)
    - `border`, `leftBorder`, `topBorder`, `bottomBorder`, `rightBorder` - границы (0/1)
    - `textPlacement` - размещение текста (Normal, Wrap)
    - `font` - ссылка на шрифт (индекс в массиве font)
  - `merge` - массив объединенных ячеек
    - `r` - номер строки (начинается с 0)
    - `c` - номер колонки (начинается с 0)
    - `w` - ширина объединения (количество колонок - 1)
    - `h` - высота объединения (количество строк - 1)
  - `font` - массив шрифтов
    - `$_faceName` - имя шрифта
    - `$_height` - размер шрифта
    - `$_bold` - жирный (true/false)
    - `$_italic` - курсив (true/false)
    - `$_underline` - подчеркивание (true/false)
    - `$_strikeout` - зачеркивание (true/false)

### Особенности структуры
- Колонки могут быть пропущены (индексы не идут подряд)
- Строки могут быть пропущены (индексы не идут подряд)
- Форматы хранятся по индексам (начинается с 1, в массиве с 0)
- Шрифты хранятся по индексам
- Объединенные ячейки описываются в массиве merge
- Параметры ячеек хранятся как `parameter` в данных ячейки
- Текст ячеек хранится в структуре `tl['v8:item']['v8:content']`

## Параметры и именованные области (важная информация от эксперта 1С)

### Параметры
Параметр — это переменная в макете, значение которой определяется в программе перед выводом документа.

**Синтаксис в макете:**
- Параметр задается в тексте ячейки в квадратных скобках: `[НомерДокумента]`, `[ДатаДокумента]`, `[Организация]`
- В XML хранится как `parameter` в данных ячейки

**Программное заполнение (BSL):**
```BSL
// Получаем область из макета
ОбластьШапка = Макет.ПолучитьОбласть("Шапка");

// Заполняем параметры (без квадратных скобок в коде!)
ОбластьШапка.Параметры.НомерДокумента = Номер;
ОбластьШапка.Параметры.ДатаДокумента = Формат(Дата, "ДЛФ=DD");
ОбластьШапка.Параметры.Организация = СсылкаНаОрганизация.Наименование;

// Выводим область в табличный документ
ТабличныйДокумент.Вывести(ОбластьШапка);
```

### Именованные области
Именованная область — это выделенный фрагмент макета (ячейка, строка, таблица), которому присвоено уникальное имя. Это ключевой механизм для циклического вывода табличных частей.

**Как создать:**
- Выделить ячейки/диапазон ячеек в макете
- Ввести имя в поле "Имя" на Панели свойств (например, `Шапка`, `Строка`, `Подвал`)

**Использование для табличных частей:**
```BSL
// Получаем область для одной строки
ОбластьСтрока = Макет.ПолучитьОбласть("Строка");

// Цикл по строкам табличной части
Для Каждого ТекСтрокаТЧ Из Товары Цикл 
    ОбластьСтрока.Параметры.Номенклатура = ТекСтрокаТЧ.Номенклатура.Наименование;
    ОбластьСтрока.Параметры.Количество   = ТекСтрокаТЧ.Количество;
    ОбластьСтрока.Параметры.Цена         = ТекСтрокаТЧ.Цена;
    ТабличныйДокумент.Вывести(ОбластьСтрока);
КонецЦикла;
```

**Важные особенности:**
- Имена областей должны быть уникальными в пределах одного макета
- При объединении ячеек текст берется из верхней левой ячейки
- При программном доступе к объединенной ячейке обращаться к верхней левой ячейке

### Панель свойств ячейки (важные свойства)
- **Текст:** Содержимое ячейки (статический текст, параметр `[ИмяПараметра]`, формула)
- **Формат:** Конструктор формата (числа, даты, логические значения)
- **Шрифт:** Гарнитура, размер, начертание (жирный, курсив), цвет
- **Цвет фона:** Заливка ячейки
- **Рамка:** Детальная настройка границ (толщина, тип линии, цвет для каждой стороны)
- **Выравнивание:** По горизонтали и вертикали
- **Ориентация:** Поворот текста на заданный угол
- **Автоподбор высоты:** Автоматическое растягивание для многострочного текста
- **Параметры разрыва:** Управление переносом строк и страниц

### Интерфейс редактора (основные элементы)
- **Панель инструментов:** Быстрый доступ к функциям (создать/удалить строку/колонку, объединить, рамки, выравнивание)
- **Область редактирования:** Рабочее поле в виде таблицы
- **Панель свойств:** Справа или снизу, отображает свойства выбранного элемента

### Как открыть редактор макетов
1. Открыть Конфигуратор
2. В дереве метаданных выбрать объект (Отчеты, Документы и т.д.)
3. Открыть объект, перейти на вкладку "Макеты"
4. Нажать "Добавить", выбрать тип "Табличный документ"
5. Нажать "Конструктор макета" или дважды кликнуть по строке макета
