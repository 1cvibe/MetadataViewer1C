## События формы (управляемая форма)

В `Form.xml` события формы задаются в блоке `<Events>`: атрибут `name` — внутреннее (английское) имя, содержимое элемента — имя процедуры-обработчика (как в модуле формы):

```xml
<Events>
  <Event name="OnCreateAtServer">ПриСозданииНаСервере</Event>
  <Event name="OnOpen">ПриОткрытии</Event>
  <Event name="BeforeClose">ПередЗакрытием</Event>
  <Event name="BeforeWrite">ПередЗаписью</Event>
  <Event name="BeforeWriteAtServer">ПередЗаписьюНаСервере</Event>
  <Event name="OnWriteAtServer">ПриЗаписиНаСервере</Event>
  <Event name="AfterWriteAtServer">ПослеЗаписиНаСервере</Event>
  <Event name="AfterWrite">ПослеЗаписи</Event>
  <Event name="OnReadAtServer">ПриЧтенииНаСервере</Event>
  <Event name="ChoiceProcessing">ОбработкаВыбора</Event>
  <Event name="NewObjectWriteProcessing">ОбработкаЗаписиНовогоОбъекта</Event>
  ...
</Events>
```

### Соответствие имён в XML (name) и обработчиков (русское имя)

| name в XML | Имя процедуры (рус.) | Описание |
|------------|----------------------|----------|
| OnCreateAtServer | ПриСозданииНаСервере | Вызывается при создании формы на сервере |
| OnOpen | ПриОткрытии | При открытии формы |
| OnReopen | ПриПовторномОткрытии | При повторном открытии формы |
| BeforeOpen | ПередОткрытием | Перед открытием формы |
| BeforeClose | ПередЗакрытием | Перед закрытием формы |
| OnClose | ПриЗакрытии | При закрытии формы |
| BeforeWrite | ПередЗаписью | На клиенте перед записью объекта из формы |
| BeforeWriteAtServer | ПередЗаписьюНаСервере | Перед записью на сервере (вне транзакции), можно отменить |
| OnWriteAtServer | ПриЗаписиНаСервере | На сервере при записи (после ПриЗаписи объекта, до конца транзакции) |
| AfterWriteAtServer | ПослеЗаписиНаСервере | После записи на сервере и завершения транзакции |
| AfterWrite | ПослеЗаписи | На клиенте после записи и окончания транзакции |
| OnReadAtServer | ПриЧтенииНаСервере | При чтении объекта на сервере |
| ChoiceProcessing | ОбработкаВыбора | При выборе в подчинённой форме / ОповеститьОВыборе |
| NewObjectWriteProcessing | ОбработкаЗаписиНовогоОбъекта | При записи нового объекта в подчинённой форме |
| ObjectActivationProcessing | ОбработкаАктивизацииОбъекта | Обработка активизации объекта |
| NotificationProcessing | ОбработкаОповещения | Обработка оповещения |
| FillCheckProcessing | ОбработкаПроверкиЗаполнения | Обработка проверки заполнения |
| RefreshDisplay | ОбновлениеОтображения | Обновление отображения |
| ExternalEvent | ВнешнееСобытие | Внешнее событие |
| AddInDetachmentOnError | ОтключениеВнешнейКомпонентыПриОшибке | Отключение внешней компоненты при ошибке |

## Поле ввода (InputField / ПолеВвода)

| Событие | Описание |
|--------|----------|
| ПриИзменении | При изменении данных в поле |
| НачалоВыбора | Перед открытием формы выбора (F4, кнопка выбора) |
| Очистка | При нажатии кнопки очистки |
| Открытие | При нажатии кнопки открытия (множественный выбор и т.п.) |
| ОбработкаВыбора | После выбора значения в форме выбора или в списке выбора |
| ИзменениеТекстаРедактирования | При наборе текста в поле ввода |
| АвтоПодборТекста | При формировании значения по отредактированному тексту |
